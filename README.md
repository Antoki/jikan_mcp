# Jikan API MCP Server

### (Work in Progress)

A MCP Server for Jikan API, to allow your favorite AI coding tool to query the MyAnimeList unofficial API and make simple analysis.

## Install the requirements (I recommend using `uv`)

`uv pip install -r requirements.txt`

## Run the server

You can use your own config file or use the schema in `mcp.example.json`. It depends of the IDE you are using.

To run the server, simply run the Python file `jikan_mcp_server.py`.

`python jikan_mcp_server.py` or `uv run jikan_mcp_server.py` if you are using uv.
